 Proyecto Final Módulo 3 - Aplicación de Consola

Esta es una aplicación interactiva desarrollada en JavaScript que se ejecuta directamente en la consola del navegador. Fue creada como evaluación para el Módulo 3, integrando conceptos de interacción, lógica matemática, arrays y objetos.

Cómo Ejecutar el Proyecto

Abrir el archivo: Localiza el archivo `index.html` en la carpeta del proyecto y ábrelo con cualquier navegador moderno (Chrome, Edge, Firefox).
Abrir la Consola:
En Windows presionar F12 o Ctrl + Shift + J.
   
Comenzar: Sigue las alertas y las instrucciones que aparecerán en la consola. 

Funcionalidades Incluidas

El proyecto demuestra el uso de:
Interacción: Captura de datos mediante `prompt` y `confirm`.
Cálculos: Operaciones matemáticas con validación de entradas.
Listas: Manipulación y filtrado de arrays con bucles.
Estructuras de Datos: Uso de objetos y métodos para organizar información.
Modularización: Código organizado en funciones reutilizables.
